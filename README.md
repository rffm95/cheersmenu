# Cheers Premium Menu

A modern, high-performance interactive menu for Cheers Bar.

## Features
- **Filter**: All items under 2 sales were removed for a cleaner experience.
- **Search**: Real-time filtering of drinks.
- **Feeling Lucky**: Random drink suggestion for indecisive customers.
- **Mobile First**: Designed specifically for smartphone use in the bar.

## Tech Stack
- HTML5 / CSS3 (Tailwind CSS)
- JavaScript (Vanilla)
- Lucide Icons
